Detailed Description

Main module.

Version
1.0 This file contains the high-level code for the project. It initialises appropriate hardware subsystems, creates application threads, and then starts the OS.

An example of two threads communicating via a semaphore is given that flashes the orange LED. These should be removed when the use of threads and the RTOS is understood.
